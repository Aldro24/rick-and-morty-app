Rick and Morty App:



La siguiente aplicación fue desarrollada con la versión de angular 18, muestra personajes con funciones como filtrado, paginación, ordenamiento y detalles de los personajes.



Funcionalidades: 

* Listado de personajes en una tabla ordenada alfabéticamente 
* Paginación con 15 datos de personajes por pagina
* Actualización al cambiar filtros



Filtros:

* Se puede ejecutar el filtrado por nombre o especie



Detalles del personaje: 

* Imagen
* Estado
* Especie
* Origen
* Localización
* Episodio en que aparece
* Personaje relacionado que sea de la misma localización



Totales: 

* Por especie
* por tipo



Tecnologías usadas: 

* Angular 18 
* TypeScript
* HTML/CSS
* RxJS
* API de Rick y Morty



Como ejecutar el proyecto: 



* Para clonar el repositorio se introduce el siguiente comando: git clone https://github.com/Aldro24/rick-and-morty-app.git
* Se instalan las dependencias con npm install
* se ejecuta con el comando ng serve
